# Yazdan Ali Khan - Portfolio Website [WIP]

Welcome to the source code for my personal portfolio website. This project showcases my background as an Electrical Engineering student, my technical skills, and my software development projects.

## Overview

This website is designed with a modern, dark-themed aesthetic featuring interactive elements to provide an engaging user experience. It serves as a central hub for my professional identity, connecting visitors to my resume, social media, and project repositories.

## Features

- **Interactive Custom Cursor**: A custom cursor system consisting of a dot, a trailing circle, and a subtle glow effect that reacts to mouse movement.
- **Hover Effects**:
  - The cursor expands and changes color when hovering over links and text.
  - Buttons and links have specific hover states for better feedback.
- **Animations**:
  - **Typing Effect**: The header text ("Yazdan Ali Khan's Portfolio Website") types itself out on load.
  - **Fade-in**: Content sections (Bio, Projects) fade in and slide up upon loading with staggered delays.
  - **Click Ripple**: A ripple animation triggers wherever the user clicks on the page.
    - **Background**: A subtle grid background with a glowing gradient effect.
- **Responsive Layout**: Built using CSS Grid and Flexbox to ensure content is aligned and accessible.

## Technologies Used

- **HTML5**: Semantic structure of the website.
- **CSS3**: Styling, animations (keyframes), and layout (Grid/Flexbox).
- **JavaScript (Vanilla)**: Logic for the custom cursor, typing animation, and click effects.
- **Font Awesome**: Icons for social media links and UI elements.

## Project Structure

- `index.html`: The main entry point containing the website structure and content.
- `style.css`: Contains all styling rules, animations, and responsive design logic.
- `script.js`: Handles the interactive behaviors (cursor movement, typing effect, click ripples).
